## REQUERIMIENTOS

Se necesita crear un proyecto para publicarlo en github pages para que el costo sea 0, para ello se utilizará una base de datos supabase.

El proyecto será una página web de albañileria, solo tendrá un usuario y la información debe de estar protegida con contraseña.

EL usuario puede agregar facturas, editarlas y eliminarlas. Además también puede generar pdfs de las facturas
